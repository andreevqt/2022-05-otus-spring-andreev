# Library app
Available commands:
```sh
# Author Commands
author:all, author:findAll, author:list: List all authors
author:create, author:insert: Create author
author:del, author:delete, author:remove: Delete an author
author:find, author:findById: Find author by id
author:update: Update an author

# Book Commands
book:all, book:findAll, book:list: List all books
book:create, book:insert: Create a new book
book:del, book:delete, book:remove: Delete a book
book:find, book:findById: Find book by id
book:update: Update a book

# Genre Commands
genre:all, genre:findAll, genre:list: List all genres
genre:create, genre:insert: Create a genre
genre:del, genre:delete, genre:remove: Delete a genre
genre:find, genre:findById: Find genre by id
genre:update: Update a genre
